## 基础用法

:::demo

textarea/basic

:::

## 禁用状态

通过`disabled`参数禁用文本域。

:::demo

textarea/disabled

:::

## 指定行数

通过`rows`参数来指定展示的行数。

:::demo

textarea/rows

:::

## 自动行数

通过`autosize`来控制是否当内容超过指定行数时，是否自动增加高度以适应内容。

:::demo

textarea/autosize

:::

## 拖动高度

通过`resize`参数控制是否允许用户拖拽来调整文本域的高度。

:::demo

textarea/resize

:::

## 长度限制

通过`maxlength`, `minlength`来限制输入的字符串长度，可以使用`show-word-count`来控制是否显示限制的字符数。

:::demo

textarea/length-limit

:::
